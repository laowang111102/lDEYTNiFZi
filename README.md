## 功能模块

系统采用B/S结构，基于JavaWeb技术开发，以MySQL为后台数据库。系统包含以下两种角色及其功能：

### 管理员

- 系统登录
- 管理员管理
- 用户管理
- 用户卡充值
- 车位管理
- 车费标准设置
- 停车缴费管理
- 车位状态查询
- 停车记录查询
- 数据导出
- 退出登录

### 用户

- 用户登录
- 修改个人信息
- 查看车辆信息
- 查询卡余额
- 查看收费标准
- 查看车位信息
- 预订车位
- 查询停车记录

## 系统角色

系统定义了两种角色：

- 管理员：负责系统的日常管理和维护。
- 用户：使用系统的普通用户，可以管理个人停车相关信息。
